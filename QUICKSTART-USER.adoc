= Game Server Admin — User Quickstart
:toc: preamble

Get up and running in 60 seconds.

== Prerequisites

* Git 2.40+
* just (command runner) — https://just.systems

== Install

[source,bash]
----
git clone https://github.com/hyperpolymath/game-server-admin
cd game-server-admin
just doctor    # check toolchain
just heal      # auto-install missing tools
----

== First Run

[source,bash]
----
just tour      # guided project tour
just help-me   # see common workflows
----

== Run the CLI

[source,bash]
----
# Start the backing database
just verisimdb-up

# Check everything is healthy
just run status

# Probe a game server
just run probe my-server.example.com 25565

# List supported games
just run profiles
----

== Get Help

* `just help-me` — common workflows
* `just doctor` — diagnose toolchain issues
* `just run help` — CLI usage
* https://github.com/hyperpolymath/game-server-admin/issues — report bugs
